# BLM RealPitch - Pitch Monitor

Real-time pitch monitor developed by Brownie Line.

BLM RealPitch is a precision tool designed by Brownie Line to help singers and musicians visualize their pitch in real time. This application offers a clear visual guide for monitoring frequency and note stability.

# BLM RealPitch - Pitch Monitor

## Main Functions

* **Real-Time Detection**: Instant identification of musical notes while singing or playing.
* **Visual Color Guide**: The monitor changes color based on your precision:
    * **Green**: Correct pitch.
    * **Orange**: Slightly deviated note (Sharp or Flat).
    * **Red**: Off-pitch.
* **Tuning History**: A moving line shows the stability of your voice over the seconds.
* **Reference Tones**: Includes a piano keyboard to hear the correct note.
* **Compatibility**: Available for Windows computers and Android devices.
